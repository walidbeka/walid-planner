# 📋 Walid Planner

مدير مهام شخصي عربي بالكامل - Mobile First - مع Firebase.

## 🚀 طريقة النشر على GitHub Pages

### 1. إنشاء مشروع Firebase

1. اذهب إلى [Firebase Console](https://console.firebase.google.com)
2. أنشئ مشروعاً جديداً
3. فعّل **Authentication** ← Google Provider
4. فعّل **Firestore Database** (قاعدة بيانات تجريبية)
5. فعّل **Functions** (للتذكير اليومي)

### 2. إعداد الملفات

1. افتح `firebase-config.js`
2. استبدل القيم التالية بمشروعك من Firebase Console:
   - `YOUR_API_KEY`
   - `YOUR_PROJECT_ID`
   - `YOUR_SENDER_ID`
   - `YOUR_APP_ID`
3. استبدل `YOUR_EMAIL@gmail.com` ببريدك الإلكتروني الشخصي

### 3. رفع على GitHub Pages

```bash
git init
git add .
git commit -m "Walid Planner"
```

ارفع المشروع إلى GitHub وشغل GitHub Pages من الإعدادات.
